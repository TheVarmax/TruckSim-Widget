TruckSim Widget Telemetry Plugin
================================

TruckSim Widget needs scs-telemetry.dll to communicate with ETS2 and ATS.

Installer users:
The setup wizard can install this plugin automatically. Select the game you want to configure and choose the game folder when asked.

Manual installation:
Copy only scs-telemetry.dll into the plugins folder of each game.

Euro Truck Simulator 2:
...\Steam\steamapps\common\Euro Truck Simulator 2\bin\win_x64\plugins

American Truck Simulator:
...\Steam\steamapps\common\American Truck Simulator\bin\win_x64\plugins

If the plugins folder does not exist, create it manually.

Do not copy the whole TruckSim Widget folder into the game folder. Only scs-telemetry.dll belongs there.
